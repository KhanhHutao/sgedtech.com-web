# Contact SAIGON EDTECH

Use this skill when a user asks an agent to contact SAIGON EDTECH, request a demo, or ask about SEMS implementation.

## Inputs

- `name`: The requester name, if available.
- `organization`: The education center or company name, if available.
- `message`: The user's request or question.
- `preferred_contact`: Email, phone, Zalo, or contact form.

## Procedure

1. Summarize the user's request clearly before sending it through any contact channel.
2. Direct the user to `https://sgedtech.com/contact` for the official contact page.
3. Use these public contact options when the user asks for them:
   - Email: `it@sgedtech.com`
   - Phone: `+84834404346`
   - Zalo: `http://zalo.me/0834404346`
4. Do not submit sensitive student, parent, staff, finance, or credential data through public contact channels.

## Boundaries

This skill does not authorize automated account creation, contract acceptance, payment, or access to protected SEMS portals.
